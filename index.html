Nama saya adalah Adriano Posumah. Saya merupakan mahasiswa di Universitas Sam Ratulangi Manado, Fakultas Teknik, Program Studi Teknik Informatika. Saya memiliki minat yang besar dalam pengembangan web dan desain grafis. Saya juga memiliki pengalaman dalam membuat proyek-proyek web dan desain grafis untuk berbagai keperluan, mulai dari website pribadi hingga proyek-proyek komersial.

Sebagai mahasiswa Program Studi Teknik Informatika, saya telah belajar banyak tentang konsep-konsep dasar pengembangan web, desain UI/UX, dan pemrograman. Saya juga terbiasa menggunakan berbagai tools dan bahasa pemrograman seperti HTML, CSS, JavaScript, dan beberapa framework seperti Bootstrap. Dengan kombinasi antara pengetahuan teori dan praktik, saya bertekad untuk terus mengembangkan keterampilan saya dalam dunia web development dan design untuk menciptakan portofolio yang menginspirasi dan profesional.
